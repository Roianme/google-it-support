# 🛠️ Technical Support Fundamentals

## 📝 Summary
I successfully completed Google's Technical Support Fundamentals course, the foundational module of the IT Support Professional Certificate. This rigorous program equipped me with the core knowledge and practical mindset required to excel in entry-level IT support roles. 

## 💡 Skills and Competencies Gained
| Module Focus | Key Skills & Knowledge Acquired |
|--------------|---------------------------------|  
| Introduction to IT | Equipped me with a solid grasp of IT fundamentals, including binary logic, system abstraction, and an understanding of the modern IT specialist's role. |
| Hardware | Provided theoritical knowledge of core hardware components from CPUs and RAM to motherboards and power supplies including their functions, interactions, and practical assembly, while also covering peripherals and mobile device fundamentals. |
| Operating Systems | Delivered a comprehensive understanding of operating system core components including process management, file systems, and the boot process. Also provided practical, hands-on experience in installing and navigating both Windows and Linux environments. |
| Networking | Equipped me with the fundamentals of computer networking, including key concepts like TCP/IP, networking hardware, and the architecture of the internet, while also exploring modern implications such as IoT, privacy, and security. |
| Software | Provided a practical understanding of the software lifecycle, from how it is built and versioned to the hands-on management of installing, updating, and removing applications on both Windows and Linux operating systems. |
| Troubleshooting |  Gained a systematic, customer-centric troubleshooting methodology. From problem isolation and documentation to developing the critical soft skills necessary for effective client interaction and conflict resolution. |
