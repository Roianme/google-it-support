# 🔤 The Bits and Bytes of Computer Networking

## 📝 Summary
This in-depth course provided me with a solid understanding of modern network infrastructure, building proficiency in core protocols like TCP/IP and DHCP, the OSI and TCP/IP models, and critical network services, while also covering essential topics in wireless networking, VPNs, and network security to ensure data integrity.

## 💡 Skills and Competencies Gained
| Module Focus | Key Skills & Knowledge Acquired |
|--------------|---------------------------------|
| Introduction to Networking | I built a solid foundation in network infrastructure by learning how data is framed and physically transmitted across networks. I gained practical knowledge of essential hardware and protocols, including the function of twisted-pair cabling, the difference between Layer 2 switches and hubs, and the role of routers. I mastered core concepts like the TCP/IP five-layer model, dissecting the structure of an Ethernet frame to understand MAC addressing. |
| The Network Layer | Begins by deconstructing IPv4 addressing, moving from classful concepts to the practical application of CIDR and subnetting to partition networks efficiently. The focus then shifts to the mechanics of data transmission, examining IP datagram encapsulation and the critical role of ARP in resolving addresses on the local link. Then the path leads to configuring routing pathways, analyzing how routing tables direct traffic and distinguishing between interior and exterior gateway protocols that govern flow across autonomous systems. |
| The Transport and Application Layers | Learned the critical functions of the transport and application layers, including the dissection of TCP segments, the role of TCP control flags in the three-way handshake for establishing reliable connections, and the distinct characteristics of connection-oriented versus connectionless protocols like UDP. I also learned how the application layer interfaces with the OSI model and gained a holistic view of how all five network layers operate in unison to facilitate end-to-end communication, including the role of firewalls in regulating traffic. |
| Network Services  | I gained comprehensive knowledge of critical network services, beginning with an in-depth understanding of DNS operations including name resolution steps, resource record types, and zone management. I also learned the DHCP lease process for dynamic IP allocation, implemented Network Address Translation (NAT) to overcome IPv4 limitations, and configured both Virtual Private Networks (VPNs) and proxy services to manage secure remote access and network traffic. |
| Connecting to the Internet | Establish deeper understanding of internet access methodologies, from foundational dial-up and T-Carrier systems to modern broadband solutions like DSL, cable, and fiber optics. This expertise extends to configuring Wide Area Networks (WANs), implementing secure wireless networks including channel management and security protocols and establishing Point-to-Point VPNs for secure remote connections. |
| Troubleshooting and the Future of Networking | This module equipped me with systematic troubleshooting techniques using command-line tools like ping, traceroute, and nslookup to isolate connectivity and DNS issues. It also provided a critical understanding of how to integrate modern technologies, including cloud service models (IaaS, PaaS, SaaS) and the IPv6 protocol, ensuring preparedness for next-generation network environments. |
